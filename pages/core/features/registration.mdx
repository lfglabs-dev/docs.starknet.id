# Domain Registration

The Starknet ID application enables users to register STARK domain names within the Starknet network. Once registered, these STARK domain become usable across all [applications](/ecosystem/integrations) that integrate Starknet ID. This seamless integration allows users to employ their registered domain names across a spectrum of Starknet applications.

## Register a domain now

1/ Navigate to the [Starknet ID application](https://app.Starknet.id/)

2/ Connect your wallet

![img](public/illustrations/connect-wallet.webp)

3/ Choose your STARK domain

![img](public/illustrations/choose-domain.webp)

4/ Complete your registration

![img](public/illustrations/complete-info.webp)

Now, users can pay in ETH or STRK.

![img](public/illustrations/pay-strk.webp)

5/ Navigate to the [Starknet ID ecosystem](/ecosystem/integrations).
